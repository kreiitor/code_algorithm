# 数据结构和算法学习项目

## 项目简介

这是一个用于学习和练习数据结构与算法的C++项目，包含了常用数据结构的实现、经典算法的实现以及相关的测试用例。

## 项目结构

```
├── data_structures/    # 数据结构实现
│   ├── Array.cpp       # 动态数组
│   ├── List.cpp        # 链表
│   ├── Stack.cpp       # 栈
│   ├── Queue.cpp       # 队列
│   ├── BinaryTree.cpp  # 二叉树
│   └── BinarySearchTree.cpp  # 二叉搜索树
├── algorithms/         # 算法实现
│   ├── Sort.cpp        # 排序算法
│   └── Search.cpp      # 查找算法
├── tests/              # 测试用例（预留）
├── docs/               # 学习文档
│   └── README.md       # 项目说明文档
└── problems/           # 经典算法题解（预留）
```

## 数据结构实现

### 1. 动态数组 (Array.cpp)
- 支持动态扩容和缩容
- 支持在任意位置插入、删除元素
- 支持添加、删除、查找元素
- 时间复杂度：
  - 访问：O(1)
  - 插入/删除（末尾）：均摊O(1)
  - 插入/删除（中间）：O(n)

### 2. 链表 (List.cpp)
- 单链表实现
- 支持虚拟头结点
- 支持添加、删除、查找、反转等操作
- 时间复杂度：
  - 访问：O(n)
  - 插入/删除（已知位置）：O(1)
  - 插入/删除（未知位置）：O(n)

### 3. 栈 (Stack.cpp)
- 基于动态数组实现
- 支持入栈、出栈、查看栈顶元素等操作
- 时间复杂度：
  - 所有操作：O(1)

### 4. 队列 (Queue.cpp)
- 基于动态数组实现
- 支持入队、出队、查看队首元素等操作
- 时间复杂度：
  - 入队：O(1)
  - 出队：O(n)（可优化为O(1)）

### 5. 二叉树 (BinaryTree.cpp)
- 基本二叉树实现
- 支持前序、中序、后序、层序遍历
- 支持计算树高、节点数量等操作

### 6. 二叉搜索树 (BinarySearchTree.cpp)
- 基于二叉树实现
- 支持插入、删除、查找等操作
- 支持查找最小值、最大值
- 时间复杂度：
  - 平均：O(log n)
  - 最坏：O(n)（退化为链表）

## 算法实现

### 1. 排序算法 (Sort.cpp)
- 冒泡排序：O(n²)
- 选择排序：O(n²)
- 插入排序：O(n²)
- 快速排序：平均O(n log n)，最坏O(n²)
- 归并排序：O(n log n)

### 2. 查找算法 (Search.cpp)
- 线性查找：O(n)
- 二分查找（迭代）：O(log n)
- 二分查找（递归）：O(log n)

## 使用方法

### 编译和运行

每个文件都包含了main函数，可以直接编译和运行：

```bash
# 编译单个文件
g++ -std=c++17 data_structures/Array.cpp -o array

# 运行编译后的程序
./array
```

### 学习路径

1. **基础数据结构**：先学习数组、链表、栈、队列等基础数据结构
2. **树结构**：学习二叉树、二叉搜索树等树结构
3. **基础算法**：学习排序算法和查找算法
4. **高级算法**：根据需要学习图算法、动态规划、贪心算法等

### 练习建议

1. 理解每个数据结构的实现原理和时间复杂度
2. 尝试自己实现数据结构和算法，然后与项目中的实现进行对比
3. 分析不同算法的优缺点和适用场景
4. 解决一些经典的算法问题，加深理解
